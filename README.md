# 🛡️ لوحة معلومات السلامة والامتثال | Safety & Compliance Dashboard

نظام شامل لتحليل بيانات السلامة والامتثال مبني باستخدام Streamlit مع تقنيات الذكاء الاصطناعي المتقدمة.

## ✨ المميزات الرئيسية

### 📊 لوحة المعلومات الرئيسية
- **مؤشرات الأداء الرئيسية (KPIs)** - عرض فوري للمقاييس الحيوية
- **نظرة عامة على الامتثال** - تتبع معدلات الامتثال في الوقت الفعلي
- **إدارة المخاطر** - تحليل وتصنيف المخاطر
- **الخرائط الحرارية للأنشطة** - تصور كثافة الأنشطة
- **تحليل الاتجاهات الزمنية** - تتبع التطورات عبر الزمن

### 🔍 التحليلات المتقدمة
- **تحليل الأداء التفصيلي** - مقارنة أداء القطاعات المختلفة
- **تحليل المخاطر المتقدم** - مصفوفات الارتباط وتوزيع المخاطر
- **التحليل التنبؤي** - توقع الاتجاهات المستقبلية
- **التحليل الموسمي** - فهم الأنماط الزمنية

### 📤 رفع البيانات اليدوي
- **دعم ملفات Excel** - رفع ومعالجة ملفات Excel متعددة الأوراق
- **دعم ملفات CSV** - رفع متعدد لملفات CSV
- **التحقق من جودة البيانات** - فحص شامل لجودة البيانات المرفوعة
- **المعاينة الفورية** - عرض البيانات قبل الدمج

### 🤖 المساعد الذكي (AI Chatbot)
- **استفسارات ذكية** - إجابات فورية على الأسئلة
- **تحليل تفاعلي** - إنشاء الرسوم البيانية حسب الطلب
- **رؤى تلقائية** - استخراج الرؤى من البيانات
- **دعم اللغة العربية** - تفاعل كامل باللغة العربية

### 📋 تقارير الجودة
- **تحليل شامل للبيانات** - إحصائيات مفصلة لكل مجموعة بيانات
- **كشف البيانات المكررة** - تحديد وإدارة التكرارات
- **تحليل أنواع البيانات** - فهم هيكل البيانات
- **تقارير الاستخدام** - مراقبة استخدام الذاكرة والأداء

## 🚀 التثبيت والتشغيل

### المتطلبات الأساسية
- Python 3.8 أو أحدث
- pip (مدير الحزم)

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone https://github.com/lokalonza/Analysis-Site.git
cd Analysis-Site
```

2. **إنشاء بيئة افتراضية (اختياري ولكن مُوصى به)**
```bash
python -m venv venv
source venv/bin/activate  # على Linux/Mac
# أو
venv\Scripts\activate  # على Windows
```

3. **تثبيت المتطلبات**
```bash
pip install -r requirements.txt
```

4. **تشغيل التطبيق**
```bash
streamlit run streamlit_app.py
```

5. **فتح المتصفح**
سيتم فتح التطبيق تلقائياً على `http://localhost:8501`
